# gestion-association
Site web dynamique de gestion d'association (PHP / MySQL)

## Installation

1. Télécharger le projet
2. Placer dans htdocs (XAMPP)
3. Importer database.sql dans phpMyAdmin
4. Aller sur :
http://localhost/association
